Tesla Stock Price Forecasting

Author: Zheng Jiang

Overview

This project analyzes and forecasts Tesla stock prices using time series models. The goal is to evaluate whether prediction intervals provide reliable coverage of future prices.

Data

Daily Tesla stock data including Open, High, Low, Close, and Volume. The dataset contains approximately 2,200 trading days.

Methodology

  • Log transformation of price series
  • Differencing to achieve stationarity
  • Model selection using ARIMA
  • Forecasting with 95% prediction intervals

Results

The ARIMA(0,1,0) with drift model provided stable forecasts. The 95% prediction intervals achieved coverage close to the expected level, indicating the model effectively captured uncertainty.

Key Insight: The model performs well for short-term forecasting but remains sensitive to market volatility.

Tools

R, forecast package
